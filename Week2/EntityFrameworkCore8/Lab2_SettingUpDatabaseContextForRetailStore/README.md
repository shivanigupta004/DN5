# Lab 2: Setting Up the Database Context for a Retail Store

## Objective
Configure a DbContext and connect it to SQL Server using EF Core.

## What you will learn
- How to create model classes for `Category` and `Product`
- How to create an `AppDbContext` using EF Core
- How to configure SQL Server in `OnConfiguring`

## Key steps
1. Define the entity models.
2. Create `AppDbContext` with `DbSet<T>` properties.
3. Configure SQL Server using `UseSqlServer`.

## Run
```powershell
dotnet run
```
