# Exercise 7 — Financial Forecasting (C#)

This project demonstrates a recursive forecasting example for a financial tool that predicts the next value based on past data.

## How it works

- `Program.cs` contains a recursive moving average forecasting algorithm.
- `Forecasting.RecursiveMovingAverageForecast` computes a forecast by recursively combining the last `k` values and the previous forecast.
- It also prints a short analysis of time and space complexity.
